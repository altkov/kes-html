Блоки распределены по файлам.
Один файл - один блок.
Но один блок - не один файл. Где-то выделены модификаторы в отдельные файлы, чтобы в файлах
не было большого количества строк.

По типу блоки распределены по папкам:
common - наиболее часто использумые блоки
slider - слайдеры
form - блоки форм или просто инпуты
layout - блоки шаблона страницы
blocks - все остальные обычные блоки

В теории какие то блоки можно перенести в директорию blocks
